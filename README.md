# Jadoo - Modern Travel Agency Landing Page

Jadoo is a visually stunning, high-performance, and responsive modern travel agency landing page built using **React**, **TypeScript**, and **Tailwind CSS**. The project features smooth animations implemented with **Framer Motion** to deliver an exceptional user experience.

---

## ✨ Features
* **Fully Responsive Design:** Optimized for mobile, tablet, and desktop screens.
* **Smooth Animations:** Fluid component entries and interactive hover effects powered by Framer Motion.
* **Clean Code Architecture:** Strongly typed components using TypeScript and modular folder structures.
* **Performance Optimized:** Clean SVG icons, optimized layouts to prevent unnecessary re-renders.

## 🛠️ Tech Stack
* **Frontend:** React 19 / 18
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** React Icons & Custom SVGs
* **Build Tool:** Vite
